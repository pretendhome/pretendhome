## 4-QUESTION FRAMEWORK (USE IN DEMO)

1. **Is there a pattern?** (Same problem across multiple users)
2. **Is there data?** (Past examples, documents, structured data)
3. **Is it repetitive?** (Happens frequently enough to justify automation)
4. **Can you measure success?** (Clear metrics: time saved, accuracy, satisfaction)

If yes to all four ‚Üí good fit for AI

## DEMO SCENARIO STRUCTURE

**Step 1: Discovery** (5-10 min)
- What does success mean to you?
- What's the current process?
- Where do users spend the most time?
- What data do you have?
- How do you measure success today?

**Step 2: Problem Framing** (5 min)
- Apply 4-question framework
- Synthesize what you heard
- Propose success metrics

**Step 3: Solution Design** (10 min)
- Input: What does the user provide?
- Retrieval: What data does the agent need?
- Generation: What does the agent produce?
- Human-in-the-loop: Where does the user review?

**Step 4: Pilot Scoping** (5 min)
- 12 weeks, 5-10 users
- Success criteria: 80%+ usage, 50%+ time savings, 4.5+ satisfaction
- Kill criteria: <50% usage, <30% time savings, <4.0 satisfaction
