# Project Structure

**File Organization**: Palette-standard structure with artifacts/ for agent outputs  
**Naming Conventions**: UPPERCASE for deliverables, lowercase for working docs  
**Documentation Standards**: Markdown for working docs, Office formats for client deliverables

## Directory Layout
- `artifacts/research/`: Argy outputs (market research, competitive analysis)
- `artifacts/architecture/`: Rex outputs (business model, financial architecture)
- `artifacts/validation/`: Anky outputs (validation reports, quality checks)
- `artifacts/narrative/`: Yuty outputs (presentations, briefings, communication)
- `remediation/`: Underwriter remediation conditions
- `garbage_collection/`: Deprecated/archived materials
- Root: Final deliverables and key documents
