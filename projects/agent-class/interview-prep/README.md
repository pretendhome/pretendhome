## Key Frameworks to Memorize

### 4-Question Method
1. Is there a pattern?
2. Is there data?
3. Is it repetitive or high-volume?
4. Can you measure success?

### Market Validation Tiers
- **HIGH**: 20+ companies, $500M+ (differentiation risk)
- **MEDIUM**: 10-20 companies, $100M-$500M (category risk)
- **LOW/GAP**: 0-5 companies, <$50M (validation risk)

### Interview Answer Structure
1. Apply 4-Question Method
2. Check Market Validation
3. Dig Deeper (Critical Questions)
4. Make Recommendation (with metrics + risks)
