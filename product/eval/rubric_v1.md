# Evaluation Rubric v1

Score each dimension from 1-5:
1. Convergence quality
2. Routing fitness
3. Actionability
4. Safety/governance
5. Uncertainty honesty
6. Expansion signal

## Pass Threshold
- Hard pass: total >= 24/30

## Additional Tracking
- One-way-door precision/recall
- Clarification-needed rate
- Novelty-only score (long-tail categories)
